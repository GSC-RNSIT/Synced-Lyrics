# Synced-Lyrics
Authored by Neetika Bishwas and Vijayendra Pandey

Team name - Synced Lyrics

Android Study Group

Basic premise of this app is to extract song name and artist of the current song that's playing on your music player. An online search for the song lyrics is done, and a notification pops up when they are found. The user can access these lyrics form within the app. The notification disappears when the song ends.
